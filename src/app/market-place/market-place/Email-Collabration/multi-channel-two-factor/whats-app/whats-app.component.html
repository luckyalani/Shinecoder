You need to create Whatsapp messages template on Twilio to use this feature. Please create a template with the following template (Only content in quotes) "Your login code for {{'{{1}}'}} is {{'{{2}}'}}"!

You can refer this template: "Your login code for ((1)) is ((2))"

{{'{{1}}'}} will be replaced with your site ex "https://abc.com

{{'{{2}}'}} will be replaced with security code ex: 123456 Your login code for https://abc.com/ is 123456
